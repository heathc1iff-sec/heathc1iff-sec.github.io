# Intel 架构与汇编基础

## 概述

要理解 AMSI 绕过技术，需要在汇编级别分析代码。本文介绍 Intel 架构和 Windows 汇编基础知识。

## 汇编语法

两种主要汇编语法：

| 语法 | 主要使用平台 | 操作数顺序 |
|------|-------------|-----------|
| Intel | Windows | 目标, 源 |
| AT&T | Linux | 源, 目标 |

**示例对比**：

```asm
; Intel 语法
mov eax, ebx    ; eax = ebx

; AT&T 语法
movl %ebx, %eax ; eax = ebx
```

本课程使用 Intel 语法。

## 内存模型

### 32 位 vs 64 位

| 特性 | 32 位 (x86) | 64 位 (x86_64) |
|------|------------|----------------|
| 地址空间 | 2GB (0 - 0x7FFFFFFF) | 128TB (0 - 0x7FFFFFFFFFFF) |
| 寄存器大小 | 32 位 | 64 位 |
| 指针大小 | 4 字节 | 8 字节 |

### 内存区域

```
┌─────────────────────────────────────┐
│              栈 (Stack)              │  ← 局部变量、返回地址
│              ↓ 向下增长              │
├─────────────────────────────────────┤
│                                     │
│              空闲空间                │
│                                     │
├─────────────────────────────────────┤
│              ↑ 向上增长              │
│              堆 (Heap)               │  ← 动态分配内存
├─────────────────────────────────────┤
│              数据段                  │  ← 全局变量
├─────────────────────────────────────┤
│              代码段                  │  ← 程序指令
└─────────────────────────────────────┘
```

**栈 (Stack)**：
- 存储静态大小的局部变量
- 存储函数返回地址
- 后进先出 (LIFO)

**堆 (Heap)**：
- 动态内存分配
- 长期持久内存
- 需要手动管理

## CPU 寄存器

### 32 位寄存器

```
┌────────────────────────────────────────────────────────────┐
│                          EAX (32位)                         │
│                    ┌───────────────────┐                   │
│                    │      AX (16位)     │                   │
│                    │   ┌─────┬─────┐   │                   │
│                    │   │ AH  │ AL  │   │                   │
│                    │   │(8位)│(8位)│   │                   │
└────────────────────┴───┴─────┴─────┴───┴───────────────────┘
```

**通用寄存器**：

| 寄存器 | 用途 |
|--------|------|
| EAX | 累加器，函数返回值 |
| EBX | 基址寄存器 |
| ECX | 计数器，循环计数 |
| EDX | 数据寄存器 |
| ESI | 源索引 |
| EDI | 目标索引 |
| EBP | 基址指针（栈帧） |
| ESP | 栈指针（栈顶） |
| EIP | 指令指针（下一条指令地址） |

### 64 位寄存器

64 位扩展了 32 位寄存器，并添加了新寄存器：

```
┌────────────────────────────────────────────────────────────┐
│                          RAX (64位)                         │
│              ┌─────────────────────────────┐               │
│              │          EAX (32位)          │               │
│              │        ┌───────────────┐    │               │
│              │        │    AX (16位)   │    │               │
│              │        │  ┌─────┬─────┐│    │               │
│              │        │  │ AH  │ AL  ││    │               │
└──────────────┴────────┴──┴─────┴─────┴┴────┴───────────────┘
```

**64 位新增寄存器**：

| 寄存器 | 说明 |
|--------|------|
| R8 - R15 | 新增通用寄存器 |
| R8D - R15D | 低 32 位 |
| R8W - R15W | 低 16 位 |
| R8B - R15B | 低 8 位 |

### 关键寄存器

| 32位 | 64位 | 用途 |
|------|------|------|
| EIP | RIP | 指令指针（下一条指令地址） |
| ESP | RSP | 栈指针（栈顶地址） |
| EBP | RBP | 基址指针（栈帧基址） |

## 函数调用约定

### 32 位 __stdcall

```
参数传递：全部通过栈
返回值：EAX
栈清理：被调用者

调用 func(1, 2, 3):
    push 3      ; 参数从右到左入栈
    push 2
    push 1
    call func
```

### 64 位 __fastcall

```
参数传递：
  - 前 4 个参数：RCX, RDX, R8, R9
  - 其余参数：栈
返回值：RAX
栈清理：调用者

调用 func(1, 2, 3, 4, 5):
    mov rcx, 1   ; 第 1 个参数
    mov rdx, 2   ; 第 2 个参数
    mov r8, 3    ; 第 3 个参数
    mov r9, 4    ; 第 4 个参数
    push 5       ; 第 5 个参数入栈
    call func
```

## 常用汇编指令

### 数据传输

```asm
mov eax, ebx      ; eax = ebx
mov eax, [ebx]    ; eax = *ebx (从内存读取)
mov [eax], ebx    ; *eax = ebx (写入内存)
lea eax, [ebx+4]  ; eax = ebx + 4 (加载有效地址)
```

### 栈操作

```asm
push eax          ; 将 eax 压入栈
pop eax           ; 从栈弹出到 eax
```

### 算术运算

```asm
add eax, ebx      ; eax = eax + ebx
sub eax, ebx      ; eax = eax - ebx
inc eax           ; eax++
dec eax           ; eax--
xor eax, eax      ; eax = 0 (清零)
```

### 函数调用

```asm
call func         ; 调用函数
                  ; 1. 将返回地址压入栈
                  ; 2. 跳转到 func

ret               ; 返回
                  ; 1. 从栈弹出返回地址
                  ; 2. 跳转到返回地址
```

### 比较和跳转

```asm
cmp eax, ebx      ; 比较 eax 和 ebx（设置标志位）
test eax, eax     ; 测试 eax 是否为 0

jmp label         ; 无条件跳转
je label          ; 相等时跳转 (ZF=1)
jne label         ; 不相等时跳转 (ZF=0)
jz label          ; 为零时跳转
jnz label         ; 非零时跳转
jg label          ; 大于时跳转（有符号）
jl label          ; 小于时跳转（有符号）
ja label          ; 大于时跳转（无符号）
jb label          ; 小于时跳转（无符号）
```

## 条件分支示例

### C 代码

```c
if (x == 0) {
    // do something
} else {
    // do other
}
```

### 汇编实现

```asm
    cmp eax, 0        ; 比较 x 和 0
    jne else_branch   ; 如果不相等，跳转到 else

    ; if 分支代码
    ...
    jmp end_if

else_branch:
    ; else 分支代码
    ...

end_if:
```

## 函数调用示例

### C 代码

```c
int result = WriteFile(handle, buffer, size, &written, NULL);
```

### 64 位汇编

```asm
    mov rcx, handle       ; 第 1 个参数
    mov rdx, buffer       ; 第 2 个参数
    mov r8, size          ; 第 3 个参数
    lea r9, [written]     ; 第 4 个参数（指针）
    push 0                ; 第 5 个参数 (NULL)
    sub rsp, 32           ; 影子空间
    call WriteFile
    add rsp, 40           ; 清理栈
    mov result, eax       ; 保存返回值
```

## 内存寻址模式

```asm
; 直接寻址
mov eax, [0x12345678]     ; 从固定地址读取

; 寄存器间接寻址
mov eax, [ebx]            ; 从 ebx 指向的地址读取

; 基址 + 偏移
mov eax, [ebx+4]          ; 从 ebx+4 地址读取

; 基址 + 索引 * 比例 + 偏移
mov eax, [ebx+ecx*4+8]    ; 数组访问
```

## 标志寄存器

```
┌─────────────────────────────────────────────────────────────┐
│  ... │ OF │ DF │ IF │ TF │ SF │ ZF │ ... │ AF │ ... │ PF │ CF │
└─────────────────────────────────────────────────────────────┘
```

**常用标志**：

| 标志 | 名称 | 说明 |
|------|------|------|
| ZF | 零标志 | 结果为 0 时置 1 |
| SF | 符号标志 | 结果为负时置 1 |
| CF | 进位标志 | 无符号溢出时置 1 |
| OF | 溢出标志 | 有符号溢出时置 1 |
