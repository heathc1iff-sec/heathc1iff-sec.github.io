# OSEP 入门指南
## 一、什么是 OSEP？
### 1.1 OSEP 简介
```plain
OSEP = Offensive Security Experienced Penetration Tester
       进攻性安全 - 高级渗透测试认证

课程代号: PEN-300
考试时长: 48 小时
考试形式: 实战渗透测试
```

### 1.2 OSEP vs OSCP
| 对比项 | OSCP (PEN-200) | OSEP (PEN-300) |
| --- | --- | --- |
| 难度 | 入门级 | 高级 |
| 重点 | 漏洞利用 | 绕过防护 |
| 环境 | 无防护 | 有杀软/白名单 |
| 技术 | 基础渗透 | 高级规避 |
| 前置要求 | 无 | 建议有 OSCP |


### 1.3 为什么学 OSEP？
```plain
现实世界的渗透测试:
├── 目标有杀软 → 需要绕过
├── 目标有白名单 → 需要绕过
├── 目标有网络过滤 → 需要绕过
├── 目标有 EDR → 需要绕过
└── OSEP 教你如何绕过这些防护
```

## 三、核心概念速览
### 3.1 什么是 Shellcode？
```plain
Shellcode = 一段可以直接执行的机器码

类比:
├── 普通程序 = 需要安装才能运行的软件
└── Shellcode = 可以直接注入内存运行的代码

用途:
├── 获取远程控制 (Reverse Shell)
├── 执行任意命令
└── 绑定端口等待连接
```

### 3.2 什么是 Dropper？
```plain
Dropper = 投递恶意代码的载体

类比:
├── Dropper = 快递包裹
└── Payload = 包裹里的东西

常见 Dropper:
├── Office 宏文档
├── HTA 文件
├── JScript 文件
└── PowerShell 脚本
```

### 3.3 什么是进程注入？
```plain
进程注入 = 把代码注入到其他进程中执行

类比:
├── 正常执行 = 在自己家里做事
└── 进程注入 = 偷偷进入别人家做事

好处:
├── 隐藏自己
├── 借用合法进程的权限
└── 绕过某些检测
```

### 3.4 什么是 AMSI？
```plain
AMSI = Antimalware Scan Interface
       反恶意软件扫描接口

作用:
├── 扫描 PowerShell 脚本
├── 扫描 VBA 宏
├── 扫描 JScript
└── 把可疑内容发给杀软检查

绕过 AMSI = OSEP 核心技能之一
```

### 3.5 什么是 AppLocker？
```plain
AppLocker = Windows 应用白名单

作用:
├── 只允许指定的程序运行
├── 阻止未授权的程序
└── 比杀软更难绕过

绕过 AppLocker = OSEP 核心技能之一
```

## 六、常见问题
### Q1: 我完全没有编程基础，能学吗？
```plain
答: 可以，但需要先补基础。

建议:
1. 先学 Python 基础 (1-2周)
2. 再学 PowerShell 基础 (1周)
3. 最后学 C 语言基础 (1-2周)
4. 然后开始 OSEP 学习
```

### Q2: 我没有 OSCP，能直接学 OSEP 吗？
```plain
答: 可以，但会比较吃力。

建议:
1. 先了解基本渗透测试概念
2. 熟悉 Metasploit 使用
3. 了解 Active Directory 基础
4. 然后开始 OSEP 学习
```

### Q3: 学习需要什么硬件？
```plain
最低配置:
├── CPU: 4 核
├── 内存: 16GB
├── 硬盘: 100GB SSD
└── 网络: 稳定的网络连接

推荐配置:
├── CPU: 8 核
├── 内存: 32GB
├── 硬盘: 256GB SSD
└── 网络: 稳定的网络连接
```

### Q4: 需要多长时间准备考试？
```plain
因人而异:

有 OSCP 基础: 2-3 个月
有渗透经验但无 OSCP: 3-4 个月
完全零基础: 4-6 个月

每天学习时间:
├── 工作日: 2-3 小时
└── 周末: 4-6 小时
```
