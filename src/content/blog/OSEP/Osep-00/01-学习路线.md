## 学习阶段划分
### 阶段一：基础准备（1-2周）
**目标**：建立必要的基础知识

| 学习内容 | 重要程度 | 预计时间 |
| --- | --- | --- |
| C 语言基础 | ⭐⭐⭐⭐ | 3天 |
| 汇编语言入门 | ⭐⭐⭐ | 2天 |
| Windows 架构概述 | ⭐⭐⭐⭐⭐ | 2天 |
| PowerShell 基础 | ⭐⭐⭐⭐⭐ | 2天 |
| C# 基础 | ⭐⭐⭐⭐ | 2天 |


**学习建议**：

+ 不需要精通，但要理解基本概念
+ 重点理解指针、内存、API 调用
+ PowerShell 和 C# 是课程核心语言

---

### 阶段二：客户端攻击（2-3周）
**目标**：掌握初始访问技术

| 模块 | 核心技能 | 重要程度 |
| --- | --- | --- |
| Office 宏攻击 | VBA、Shellcode Runner | ⭐⭐⭐⭐⭐ |
| 日历钓鱼 | ICS、Responder | ⭐⭐⭐ |
| Windows Script Host | Jscript、DotNetToJscript | ⭐⭐⭐⭐ |
| 反射式 PowerShell | 内存执行、反射 | ⭐⭐⭐⭐⭐ |


**核心代码清单**：

1. VBA Shellcode Runner
2. PowerShell Shellcode Runner
3. Jscript Dropper
4. DotNetToJscript 模板
5. HTML Smuggling 模板

---

### 阶段三：进程操作（1-2周）
**目标**：掌握进程注入技术

| 技术 | 核心 API | 重要程度 |
| --- | --- | --- |
| 进程注入 | OpenProcess, VirtualAllocEx, WriteProcessMemory, CreateRemoteThread | ⭐⭐⭐⭐⭐ |
| DLL 注入 | LoadLibrary, GetProcAddress | ⭐⭐⭐⭐ |
| 反射式 DLL 注入 | PE 解析、内存映射 | ⭐⭐⭐⭐ |
| 进程镂空 | NtUnmapViewOfSection, SetThreadContext | ⭐⭐⭐⭐⭐ |


**核心代码清单**：

1. C# 进程注入器
2. DLL 注入器
3. 进程镂空实现

---

### 阶段四：防护绕过（2-3周）
**目标**：掌握各种绕过技术

| 模块 | 核心技术 | 重要程度 |
| --- | --- | --- |
| 杀软绕过 | 签名绕过、混淆 | ⭐⭐⭐⭐⭐ |
| AMSI 绕过 | 内存补丁、反射 | ⭐⭐⭐⭐⭐ |
| AppLocker 绕过 | LOLBAS、CLM 绕过 | ⭐⭐⭐⭐ |
| 网络过滤绕过 | 域前置、DNS 隧道 | ⭐⭐⭐ |


**核心代码清单**：

1. AMSI 绕过脚本
2. AppLocker 绕过技术
3. PowerShell CLM 绕过
4. DNS 隧道客户端

---

### 阶段五：后渗透与横向移动（2-3周）
**目标**：掌握内网渗透技术

| 模块 | 核心技术 | 重要程度 |
| --- | --- | --- |
| Linux 后渗透 | 共享库劫持 | ⭐⭐⭐ |
| Kiosk 突破 | 浏览器逃逸 | ⭐⭐ |

| 横向移动 | WMI、WinRM、DCOM | ⭐⭐⭐⭐⭐ |


**核心代码清单**：

1. Mimikatz 使用命令
2. WMI 横向移动脚本
3. WinRM 横向移动脚本
4. DCOM 横向移动脚本

---

### 阶段六：Active Directory 攻击（3-4周）
**目标**：掌握 AD 高级攻击技术
