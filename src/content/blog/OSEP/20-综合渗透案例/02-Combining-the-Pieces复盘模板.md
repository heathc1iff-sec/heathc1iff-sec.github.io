## 决策分支

| 卡住的位置 | 先确认 | 下一步选择 |
|---|---|---|
| 初始入口失败 | 端口、Web 功能、上传限制、文件类型、代理/出网 | 换入口：Office/日历/JScript、MSSQL、弱凭证、ADCS 枚举 |
| Web Shell 不稳定 | 权限、架构、杀软、代理、当前用户上下文 | 先做最小命令和出网验证，再换更小的加载器 |
| 提权失败 | 当前权限、可用特权、补丁、服务账户、可写路径 | 转凭证、Token、计划任务、服务配置、MSSQL/AD 路径 |
| LSASS/PPL 受阻 | 是否有本地管理员/SYSTEM，Defender 状态，PPL 状态 | 转 MiniDump 离线、Token、RDP 凭证、机器账户路径 |
| S4U/委派失败 | SPN、目标服务名、票据可转发、时间同步、机器账户 Hash | 转 ADCS、ACL、RBCD、MSSQL 链接服务器 |
| 横向移动失败 | 端口、认证格式、UAC、服务权限、防火墙、出网 | 换 RDP/WinRM/SMB/服务控制/Fileless，或先做代理 |
| 域管会话未出现 | 哪些主机有管理员活动、会话是否可见、Token 类型 | 回到枚举循环：文件服务器、管理机、JIT/JEA、SQL、Linux/DevOps |
