# 面向对象编程基础

## 什么是面向对象编程？

**面向对象编程**（Object-Oriented Programming，OOP）是一种编程范式，将数据和操作数据的方法组织成"对象"。

**为什么渗透测试要了解 OOP？**
- C#、Java、PowerShell 都是面向对象语言
- 理解 OOP 才能读懂和修改攻击工具的代码
- 逆向分析时需要理解类的结构

---

## 类（Class）

### 什么是类？

**类**是创建对象的"模板"或"蓝图"。

**类比**：
- 类 = 汽车设计图纸
- 对象 = 根据图纸造出的具体汽车

### 类的基本结构

```csharp
public class Car
{
    // 变量（属性）- 存储数据
    private string brand;
    private int speed;

    // 方法 - 执行操作
    public void Accelerate()
    {
        speed += 10;
    }
}
```

**组成部分**：
| 组成 | 作用 | 示例 |
|------|------|------|
| 变量/字段 | 存储数据 | `private string brand;` |
| 方法 | 执行操作 | `public void Accelerate()` |
| 构造函数 | 初始化对象 | `public Car()` |

---

## 对象（Object）

### 什么是对象？

**对象**是类的具体实例。

**创建对象的过程叫"实例化"**：

```csharp
// 类定义
public class Car
{
    public string Brand;
}

// 实例化对象
Car myCar = new Car();      // myCar 是一个对象
myCar.Brand = "Toyota";

Car yourCar = new Car();    // yourCar 是另一个对象
yourCar.Brand = "Honda";
```

**要点**：
- 一个类可以创建多个对象
- 每个对象有自己独立的数据
- 对象之间互不影响

## 构造函数（Constructor）

### 什么是构造函数？

**构造函数**是创建对象时自动调用的特殊方法，用于初始化对象。

**特点**：
- 名称与类名相同
- 没有返回类型
- 创建对象时自动执行

### 示例

```csharp
public class Shellcode
{
    private byte[] payload;
    private int size;

    // 构造函数
    public Shellcode(byte[] data)
    {
        this.payload = data;
        this.size = data.Length;
        Console.WriteLine("Shellcode 已初始化，大小: " + size);
    }
}

// 使用
byte[] buf = new byte[] { 0x90, 0x90, 0xCC };
Shellcode sc = new Shellcode(buf);  // 自动调用构造函数
```

**输出**：
```
Shellcode 已初始化，大小: 3
```

## 访问修饰符

### 什么是访问修饰符？

**访问修饰符**控制类、方法、变量的可见性。

### 常用修饰符

| 修饰符 | 含义 | 可访问范围 |
|--------|------|-----------|
| `public` | 公开 | 任何地方都能访问 |
| `private` | 私有 | 只有类内部能访问 |
| `protected` | 受保护 | 类内部和子类能访问 |
| `internal` | 内部 | 同一程序集内能访问 |

### 示例

```csharp
public class Injector
{
    public string TargetProcess;     // 外部可以访问
    private IntPtr processHandle;    // 只有类内部能访问

    public void Inject()             // 外部可以调用
    {
        OpenProcess();               // 内部调用私有方法
    }

    private void OpenProcess()       // 外部无法调用
    {
        // 打开进程的代码
    }
}

// 使用
Injector inj = new Injector();
inj.TargetProcess = "notepad";  // OK，public
inj.Inject();                   // OK，public
// inj.processHandle = ...      // 错误！private 无法访问
// inj.OpenProcess();           // 错误！private 无法访问
```

### 为什么要用 private？

**封装**：隐藏内部实现细节，只暴露必要的接口。

**好处**：
- 防止外部代码意外修改内部数据
- 可以自由修改内部实现而不影响外部代码
- 代码更安全、更易维护

## 方法（Method）

### 什么是方法？

**方法**是类中定义的函数，用于执行特定操作。

### 方法的组成

```csharp
public int Add(int a, int b)
{
    return a + b;
}
```

| 部分 | 示例 | 说明 |
|------|------|------|
| 访问修饰符 | `public` | 谁能调用这个方法 |
| 返回类型 | `int` | 方法返回什么类型的值 |
| 方法名 | `Add` | 方法的名称 |
| 参数 | `(int a, int b)` | 方法需要的输入 |
| 方法体 | `{ return a + b; }` | 方法的具体代码 |

### 特殊返回类型：void

`void` 表示方法不返回任何值：

```csharp
public void PrintMessage(string msg)
{
    Console.WriteLine(msg);
    // 没有 return 语句
}
```

## 静态成员（Static）

### 什么是静态成员？

**静态成员**属于类本身，而不是某个具体对象。

**特点**：
- 不需要创建对象就能使用
- 所有对象共享同一份静态成员
- 用 `static` 关键字声明

### 示例

```csharp
public class Counter
{
    public static int Count = 0;  // 静态变量

    public static void Increment()  // 静态方法
    {
        Count++;
    }
}

// 使用（不需要创建对象）
Counter.Increment();
Counter.Increment();
Console.WriteLine(Counter.Count);  // 输出: 2
```

### 渗透测试中的应用

很多攻击工具使用静态方法作为入口点：

```csharp
public class Payload
{
    public static void Main(string[] args)  // 程序入口点
    {
        Execute();
    }

    public static void Execute()
    {
        // 执行 shellcode
    }
}
```

## this 关键字

### 什么是 this？

`this` 指向当前对象自身。

### 用途

**1. 区分同名变量**：

```csharp
public class Person
{
    private string name;

    public Person(string name)
    {
        this.name = name;  // this.name 是字段，name 是参数
    }
}
```

**2. 在方法中引用当前对象**：

```csharp
public class Injector
{
    public Injector GetSelf()
    {
        return this;  // 返回当前对象
    }
}
```

## 实战示例：简单的进程注入器类

```csharp
using System;
using System.Runtime.InteropServices;

public class SimpleInjector
{
    // 私有变量
    private string targetProcess;
    private byte[] shellcode;

    // 构造函数
    public SimpleInjector(string process, byte[] payload)
    {
        this.targetProcess = process;
        this.shellcode = payload;
    }

    // 公开方法
    public void Inject()
    {
        Console.WriteLine("目标进程: " + targetProcess);
        Console.WriteLine("Shellcode 大小: " + shellcode.Length);
        // 实际注入代码...
    }

    // 静态工厂方法
    public static SimpleInjector Create(string process, byte[] payload)
    {
        return new SimpleInjector(process, payload);
    }
}

// 使用方式1：直接实例化
byte[] sc = new byte[] { 0x90, 0x90 };
SimpleInjector inj1 = new SimpleInjector("notepad", sc);
inj1.Inject();

// 使用方式2：静态工厂方法
SimpleInjector inj2 = SimpleInjector.Create("calc", sc);
inj2.Inject();
```

## 本节要点总结

| 概念 | 要点 |
|------|------|
| 类 | 创建对象的模板 |
| 对象 | 类的具体实例 |
| 构造函数 | 初始化对象，与类同名 |
| public | 公开，任何地方可访问 |
| private | 私有，仅类内部可访问 |
| static | 静态，属于类而非对象 |
| this | 指向当前对象 |
