# 前置知识 - 新手学习指南

## 这个目录是干什么的？

这是你的"地基"。就像盖房子要先打地基一样，学渗透测试也需要先掌握这些基础知识。

---

## 通俗理解这些知识

### C语言基础 - 为什么要学？

```
想象一下：
├── 你想控制一台电脑
├── 电脑只听得懂"机器语言"
├── C语言是最接近机器语言的高级语言
└── 学会C语言 = 学会和电脑底层对话

你需要掌握的核心概念：
├── 变量 = 存东西的盒子
├── 指针 = 盒子的地址（门牌号）
├── 函数 = 一套固定的操作步骤
└── 内存 = 电脑的"工作台"
```

### 汇编语言入门 - 为什么要学？

```
想象一下：
├── C语言是"简化版"的指令
├── 汇编语言是"原始版"的指令
├── Shellcode 就是用汇编写的
└── 不需要精通，能看懂就行

你需要掌握的核心概念：
├── 寄存器 = CPU的临时存储格子（EAX, EBX, ECX...）
├── 栈 = 一摞盘子（后进先出）
├── PUSH/POP = 放盘子/拿盘子
└── CALL/RET = 调用函数/返回
```

### Windows架构概述 - 为什么要学？

```
想象一下：
├── Windows 是一座大楼
├── 你想在大楼里"搞事情"
├── 必须先了解大楼的结构
└── 知道哪里是入口、哪里是监控

你需要掌握的核心概念：
├── 进程 = 正在运行的程序（一个个房间）
├── 线程 = 房间里干活的工人
├── API = 大楼提供的服务（电梯、空调）
└── 内核 = 大楼的控制中心
```

### PowerShell基础 - 为什么要学？

```
想象一下：
├── PowerShell 是 Windows 的"超级命令行"
├── 它能做几乎所有事情
├── 而且很多杀软不会拦截它
└── 是渗透测试的核心工具

你需要掌握的核心概念：
├── Cmdlet = 内置命令（Get-Process, Set-Item...）
├── 管道 | = 把一个命令的结果传给下一个
├── 变量 $ = 存储数据
└── 脚本 .ps1 = 一系列命令的集合
```

## 推荐学习顺序

```
第1天-第2天：PowerShell基础
    ↓ （最实用，马上能用）
第3天-第4天：Windows架构概述
    ↓ （理解你要攻击的目标）
第5天-第6天：C语言基础
    ↓ （理解底层代码）
第7天：汇编语言入门
    （了解即可，不用深入）
```

## 实操环境搭建

### 你需要准备的虚拟机

```
最简配置：
├── VMware Workstation 或 VirtualBox（免费）
├── Windows 10/11 虚拟机（用于练习）
└── 内存至少分配 4GB

下载地址：
├── VMware: https://www.vmware.com/products/workstation-player.html
└── Windows 10 开发者虚拟机: https://developer.microsoft.com/windows/downloads/virtual-machines/
```

### 环境配置步骤

```
1. 安装 VMware/VirtualBox
2. 导入 Windows 虚拟机
3. 在虚拟机中安装：
   ├── Visual Studio Code（代码编辑器）
   ├── Visual Studio Community（C#开发）
   └── 启用 PowerShell 执行策略
```

## 实践任务清单

### PowerShell 练习（

```powershell
# 任务1：列出所有正在运行的进程
Get-Process

# 任务2：找到名为 notepad 的进程
Get-Process -Name notepad

# 任务3：获取系统信息
Get-ComputerInfo

# 任务4：写一个简单脚本
# 创建文件 test.ps1，内容：
Write-Host "Hello, Hacker!"
$name = Read-Host "你的名字"
Write-Host "欢迎你, $name"
```

### Windows 架构练习

```powershell
# 任务1：查看进程和线程
Get-Process | Select-Object Name, Id, Threads

# 任务2：查看进程的模块（DLL）
Get-Process notepad | Select-Object -ExpandProperty Modules

# 任务3：查看服务
Get-Service | Where-Object {$_.Status -eq "Running"}

# 任务4：查看注册表
Get-ItemProperty "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Run"
```

### C语言练习

```c
// 任务1：理解指针
// 在 Visual Studio 中创建一个 C 项目

#include <stdio.h>

int main() {
    int number = 42;
    int *pointer = &number;  // 指针存储 number 的地址

    printf("number 的值: %d\n", number);
    printf("number 的地址: %p\n", &number);
    printf("pointer 存储的地址: %p\n", pointer);
    printf("pointer 指向的值: %d\n", *pointer);

    return 0;
}

// 任务2：理解函数调用
// 调用 Windows API
#include <windows.h>
#include <stdio.h>

int main() {
    // 弹出一个消息框
    MessageBox(NULL, "Hello from C!", "测试", MB_OK);
    return 0;
}
```

## 学习检验

完成以下问题，检验你的学习成果：

```
□ 能解释什么是指针吗？
□ 能说出 EAX、ESP、EIP 寄存器的作用吗？
□ 能解释进程和线程的区别吗？
□ 能写一个简单的 PowerShell 脚本吗？
□ 知道什么是 Win32 API 吗？
```
