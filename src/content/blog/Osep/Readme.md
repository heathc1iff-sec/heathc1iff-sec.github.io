# PEN-300 OSEP 知识库

## PEN-300 OSEP 中文入门指南

---

## 关于本知识库

本知识库基于 OffSec 官方 PEN-300 (OSEP - Offensive Security Experienced Penetration Tester) 课程创建

### 学习目标

通过本知识库的学习，你将能够：

1. **理解 Windows 底层机制** - 从 Web 安全视角理解 Windows 内部工作原理
2. **掌握客户端攻击技术** - Office 宏、Jscript、HTML Smuggling 等
3. **学会绕过安全防护** - 杀软绕过、AMSI 绕过、AppLocker 绕过
4. **精通横向移动技术** - Windows/Linux 环境下的横向移动
5. **深入 Active Directory 攻击** - Kerberos 委派、域信任攻击等高级技术

## 如何使用本知识库

### 1. 顺序学习
按照模块顺序从头到尾学习，每个模块都有：
- **概念讲解**：用通俗易懂的语言解释技术原理
- **代码示例**：带详细注释的代码
- **实战演练**：动手练习步骤
- **章节测试**：检验学习效果

### 2. 查漏补缺
如果你已有一定基础，可以直接跳到感兴趣的章节学习。

### 3. 考试准备
考试前重点复习标记为 ⭐⭐⭐⭐⭐ 的内容。

## 写给 Web 安全人员的话

作为一个 Web 安全从业者，你可能对以下概念感到陌生：

| Web 安全概念 | 对应的二进制/系统概念 |
|-------------|---------------------|
| XSS 注入 | Shellcode 注入 |
| SQL 注入 | 内存注入 |
| 文件上传 | DLL 注入 |
| 会话劫持 | Token 窃取 |
| 权限绕过 | UAC 绕过 |
| WAF 绕过 | 杀软绕过 |

**好消息是**：很多概念是相通的，只是作用的层面不同。Web 安全在应用层，而 OSEP 的内容更多在系统层。
